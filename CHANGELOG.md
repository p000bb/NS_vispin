# 1.0.0
- 初版发布

# 1.1.0
Features
- 添加自动更新功能

# 1.1.1
Features
- 新增系统托盘
- 优化引脚功能下拉框的长度逻辑，避免出现溢出问题
- 优化底部引脚文字显示的方向，调整为和上方引脚文字显示方向一致

Fixes
- 修复导出Excel的时候，缺少自定义EXTI列
- 修复解析芯片数据的时候，引脚数据中包含空格导致引脚功能冲突检测失效的问题      
- 修复部分组合引脚的EXTI功能无法正常显示的问题

# 1.1.2
Features
- 优化引脚名称的显示，现在会自动根据内容自适应文字大小

Fixes
- 修复统计面板在切换芯片数据的时候，引脚数据没有更新导致统计数据不正确的问题
- 修复引脚功能冲突检测在切换芯片数据的时候，引脚数据没有更新导致检测结果不正确的问题

# 1.1.3
Features
- 新增引脚功能悬浮提示功能，鼠标悬浮在引脚上时，会显示引脚功能

Fixes
- 修复程序路径在中文路径下无法正常更新程序的问题

# 1.1.4
Features
- 新增支持EXTI二级菜单选择，优化EXTI的展现形式

# 2.0.0
Features
- 新增BGA格式芯片支持
  
# 2.0.1
Features
- 新增BGA格式下引脚间距比例配置
- 优化BGA格式字母升序展现形式，按照行业标准默认跳过部分字母，并且支持双字母展示
- 优化BGA格式引脚选中文本的字体大小，根据芯片大小和脚间距比例配置自调整

Fixes
- 修复ctrl_z在部分情况下撤回导致芯片置空的问题


# 2.0.2
Features
- 完善引脚功能悬浮文本介绍功能

Fixes
- 修复引脚配置颜色切换卡顿的问题

# 2.0.3
Features
- 完善芯片引脚起始位置标记点的样式，添加BGA格式的引脚起始位置标记点支持
- 优化在高引脚数下，芯片配置字体大小比例和引脚间距比例计算卡顿的问题
- 优化在高引脚数下，PNG、PDF下载模糊的问题

# 2.0.4
Fixes
- 修复EXTIx中断取值逻辑，其中x由之前读取引脚名称最后一组数字改成读取引脚名称第一组数字
- 修复引脚下拉列表功能包含空格的问题，目前会去除所有的空格
- 修复同一芯片下，存在多个同名引脚时，导致Excel数据重复的问题

# 2.0.5
Features
- 新增批量删除项目功能

Fixes
- 修复NRST引脚功能无法选择的问题
- 修复EXTIx取值问题，现在会准确的读取引脚名称中的数字
