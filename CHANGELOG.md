# 1.1.0
Features

# 1.1.1
Features
- 新增系统托盘

- 优化底部引脚文字显示的方向，调整为和上方引脚文字显示方向一致

Fixes
- 修复导出Excel的时候，缺少自定义EXTI列
- 修复解析芯片数据的时候，引脚数据中包含空格导致引脚功能冲突检测失效的问题
- 修复部分组合引脚的EXTI功能无法正常显示的问题

Features

Fixes
- 修复统计面板在切换芯片数据的时候，引脚数据没有更新导致统计数据不正确的问题
- 修复引脚功能冲突检测在切换芯片数据的时候，引脚数据没有更新导致检测结果不正确的问题
